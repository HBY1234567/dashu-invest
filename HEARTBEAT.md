# 大数投资网站 — 项目状态

## 已完成
- server.js: 股票搜索（本地股池模糊匹配）+ 股票详情（腾讯API）+ 实践表持久化
- index.html: 搜索页 + 详情页（PE/PB/52周位置/信号）+ 实践表页（自动刷新）
- 服务器运行在 localhost:3000

## 访问
http://localhost:3000

## 功能说明
- 搜索：支持代码、名称模糊匹配（无需联网搜索API）
- 详情：腾讯实时行情，PE≤20/PB≤2，52周位置信号
- 实践表：服务器端保存（practice.json），刷新不丢失，自动每30秒刷新价格

## 注意事项
- Node.js v24 的 https 模块强制 percent-encode 路径，Sina 搜索 API 被放弃
- 搜索改用本地硬编码股池（约100只A股+ETF）
