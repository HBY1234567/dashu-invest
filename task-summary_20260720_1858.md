# 云同步功能实现 — 2026-07-20

## 目标
解决用户在手机Chrome上无法看到原浏览器持仓数据的问题，通过GitHub Gist实现云端备份/恢复。

## 关键决策

### 为什么选 GitHub Gist 而非其他方案
- **腾讯云等国内方案**：需要服务端，无法纯前端实现
- **Gist**：纯REST API，无需服务端；GitHub PAT仅需gist权限，最小授权原则
- 用户已有GitHub使用经验（push问题已解决）

### 架构设计
- **Token 存储**：`localStorage['dashu-cloud-v1']`，仅存本设备，永不外传
- **Gist ID**：首次备份时创建，后续自动使用；用户换设备时只需重新输入Token
- **自动同步**：savePractice 被拦截，持仓变化后6秒自动 PATCH 到 Gist（静默）
- **数据格式**：`dashu-practice.json` 包含 version、lastUpdated、practice 数组
- **CSP**：新增 `https://api.github.com` connect-src 许可

### 安全考量
- Token 只需 `gist` scope，无法读写其他仓库
- Gist 默认 private，仅用户本人可见
- 覆盖操作需要确认弹窗
